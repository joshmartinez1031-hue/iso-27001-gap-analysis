# ISO 27001 Gap Analysis Project

## Overview
This project demonstrates an ISO/IEC 27001 gap analysis conducted on a mock financial services organization (75 employees). The objective was to assess the organizationâ€™s current security posture against selected Annex A controls and identify compliance gaps.

## Scope
- 8 ISO 27001 Annex A controls reviewed
- Current state assessment performed
- Gaps identified and categorized
- Risk levels assigned (High / Medium)
- Remediation recommendations provided
- Audit evidence requirements documented

## Key Findings
- Total Controls Reviewed: 8
- High Risk Gaps: 5
- Medium Risk Gaps: 3
- Fully Implemented Controls: 0
